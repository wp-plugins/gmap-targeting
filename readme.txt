Set google map in post, page or everywere on your site by shortcode.

== Description ==

Set google map everywere on your site using shortcodes like [gmap_targeting post_id=2] in post
or <?=do_shortcode("[gmap_targeting post_id=2]") ?> in wp theme.
Each map belongs to post or page. To start using this plugin simply install it. Now, in pages/posts appears panel "Google Map Targeting".
After putting a geographical point, map size, map type, map position in the text,
the text in the map info window - press the update button to save post/page. That is all ....


== Installation ==

= New Installations =
0.      Download plugin and then go by link http://your_site/wp-admin/plugin-install.php?tab=upload
	press "Install a plugin in .zip format", select gmap-targeting.zip, press "Install Now".

OR

1.	Download plugin archive in zip or gzipped tar format and
	extract the files on your computer.

2.	Create a new directory named `gmap-targeting` in the `wp-content/plugins`
	directory of your WordPress installation. Use an FTP or SFTP client to
	upload the contents of plugin archive to the new directory
	that you just created on your web host.

3.	Log in to the WordPress http://your_site/wp-admin/plugins.php and activate the plugin.

4.	Configure the plugin and add it to the plugin settings page.
